User’s GuideDescription.This application counts the frequency of each letter on a message. This program is useful for cryptanalysis.Files enclosed for demonstration.The folder containing this program in the repository, has a text file called Message1.txt. This text file can be used to demonstrate the functionality of the program	User’s guide.In order to execute the program is necessary to compile and run it through command line. For executing it, it is necessary to input an argument through command line.
